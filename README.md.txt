# Proyecto U juego batalla naval

![Logo del Proyecto](img/imagen.png)

## Descripción

Este proyecto es una aplicación desarrollada en Unreal Engine 5 y C++, “Battle ship” es un juego de estrategia en el que el jugador coloca una serie de barcos en el tablero. El objetivo principal del juego es descubrir y hundir los barcos del oponente antes de que este hunda los propios.
La aplicación está construida utilizando las siguientes tecnologías:

- **Frontend**: Unreal Engine 5.3
- **Backend**: C++
- **Base de Datos**: MySql

## Características

- Añadir, editar y eliminar tareas
- Marcar tareas como completadas
- Filtrar tareas por estado (completadas, pendientes)
- Interfaz de usuario intuitiva y responsiva

Sigue estos pasos para instalar y ejecutar el proyecto en tu máquina local:

1. Clona el repositorio:
    ```bash
    git clone https://github.com/usuario/nombre-del-proyecto.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd nombre-del-proyecto
    ```
3. Instala las dependencias del servidor:
    ```bash
    Unreal Engine
    Visual studio 2022 para C++
    ```
    ```

## Uso

Una vez que la aplicación esté en funcionamiento, ejecute el debuguer de Unreal Engine 5.
